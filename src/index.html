Chicago Free-Food Network Database

Built a relational SQL database in Microsoft Access to support a nonprofit’s operations during a SNAP benefits pause. The system centralizes donor, inventory, recipient, and distribution data, and includes custom entry forms, automated queries, and formatted reports for grant tracking and resource planning.
